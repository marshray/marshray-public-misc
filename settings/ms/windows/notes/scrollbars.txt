suggests scrollbar width can be changed

### Win-R -> cmd.exe
```CMD
reg query "HKCU\Control Panel\Desktop" /v CaretWidth
```

### Win-R -> cmd.exe
```CMD
reg query "HKCU\Control Panel\Mouse" /v MouseTrails
rem reg add "HKCU\Control Panel\Mouse" /v MouseTrails /t REG_SZ /d "7" /f
```

reg query "HKCU\Control Panel\Desktop\WindowMetrics"
